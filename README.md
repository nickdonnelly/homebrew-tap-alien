# Nickdonnelly Tap-alien

## How do I install these formulae?

`brew install nickdonnelly/tap-alien/<formula>`

Or `brew tap nickdonnelly/tap-alien` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "nickdonnelly/tap-alien"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
